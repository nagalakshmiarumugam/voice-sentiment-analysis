# 🎙️ Voice-Based Sentiment Analysis System

This project performs sentiment analysis on speech using AI models.

---

## 🚀 Project Workflow

Audio File  
↓  
Whisper Model (Speech to Text)  
↓  
DistilBERT Model (Sentiment Analysis)  
↓  
Generate Response  
↓  
Google Text-to-Speech (Audio Output)

---

## 🧠 Technologies Used

- Python  
- OpenAI Whisper  
- HuggingFace Transformers (DistilBERT)  
- Google Text-to-Speech (gTTS)  
- PyTorch  
- Google Colab  

---

## 📌 Features

- Upload audio file (.mp3 / .wav)
- Convert speech to text
- Detect sentiment (Positive / Negative)
- Generate automatic response
- Convert response to speech audio

---

## 🧪 Example

### Input:
"I am very happy with this product"

### Output:
Sentiment: POSITIVE  
Response: Thank you for your positive feedback  
Audio output generated 🎧

---

## ▶️ How to Run

1. Open Google Colab
2. Upload notebook
3. Install dependencies
4. Run step-by-step cells
5. Upload audio file and get output
